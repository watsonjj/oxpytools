Oxford GCT Python Tools (oxpytools)¶

version: 0.0.dev246

What is oxpytools?¶

oxpytools is a python package that utilises both ctapipe and targetio for processing of both simulation data and GCT data.

  • Code, feature requests, bug reports, pull requests: https://github.com/watsonjj/oxpytools
  • Docs: https://watsonjj.github.io/oxpytools
  • License: BSD-3
  • Python 3.5 or later (Python 2 is not supported)
